WooCommerce Weight Based Inventory gives you the best inventory management platform for simple products based on their weight.

== Description ==

WooCommerce Weight Based Inventory allows you to manage stock for simple products based on their weight. 

Plugin is compatible with WooCommerce 3.0+ and is tested on 3.3.1 version. 

== Features of the plugin ==

*   <strong> Free version supports only Simple Products </strong>
*   Your customer can enter their required weight and we will calculate the price for the required weight.
*   You can create both weight based inventory and WooCommerce Quantity based inventory for different simple products.
*   You can set "Notification" for Low Stock and Out of Stock for weight based inventory.
*   You can set "Notification Recipient(s)" for sending Email on threshold reach.
*   You can set "Stock Threshold" for the notification.
*   You can set "Stock Format" for customizing the display format on single product page.
*   You can set "Order Actions" for each statuses whether the weight has to reduce or not.
*   You can do "Quick Edit" of all weight based inventory simple products from a single page.

For complete list of settings documentation, please visit <a rel="nofollow" href="https://stepqueue.com/documentation/woocommerce-weight-based-inventory-setup/">How To Setup WooCommerce Weight Based Inventory?</a>

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'WooCommerce Weight Based Inventory'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `woo-weight-based-inventory.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `woo-weight-based-inventory.zip`
2. Extract the `woo-weight-based-inventory` directory to your computer
3. Upload the `woo-weight-based-inventory` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
